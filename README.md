# PhD
Hi! I'm Manuel Torres-Godoy, a PhD student at the University of Southampton specializing in physical oceanography.

My PhD research will focus on the role of mixing in the Subpolar North Atlantic region using observations, aligned with the POLEMIX project's objectives.

I did my masters in Universidad de Concepcion (Chile), where investigated deep current variability in the Southeast Pacific region using both models and observations.

This repository will be used for storing the code developed for my thesis project.
